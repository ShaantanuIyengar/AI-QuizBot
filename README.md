# AI QuizBot

AI QuizBot is a Python-based quiz generator that uses the OpenAI API to create custom quiz questions in Maths or Science at different difficulty levels.

The bot:
- Generates questions using AI  
- Checks your answers  
- Provides explanations  
- Includes a Demo Mode if no API key is found  


## Features

- AI-generated quiz questions  
- Difficulty selection (easy, medium, hard)  
- Subjects: Maths or Science  
- AI evaluation of your answers  
- Explanation provided for each question  
- Optional Demo Mode  
- Secure API key handling with `.env`
